
# Проект: Персонализированные предложения для клиентов интернет-магазина «В один клик»

## Постановка задачи:
Интернет-магазин «В один клик» заметил снижение активности своих покупателей. Привлечение новых клиентов уже не так эффективно, поэтому решено удерживать активность постоянных клиентов с помощью персонализированных предложений. Для этого необходимо разработать модель, которая будет предсказывать вероятность снижения покупательской активности клиента в следующие три месяца.

## Используемые данные:
В проекте используются следующие данные:
- Признаки коммуникации сотрудников компании с клиентами.
- Продуктовое поведение покупателей (например, какие товары покупали и как часто).
- Покупательское поведение (например, сколько денег тратили в магазине).
- Поведение на сайте (например, сколько страниц просматривали и сколько времени проводили на сайте).
- Финансовые данные о доходе от каждого клиента за последние три месяца.

## Решаемая задача:
1. Построить модель, предсказывающую вероятность снижения покупательской активности клиентов.
2. Использовать результаты модели для сегментации клиентов.
3. Разработать персонализированные предложения для каждого сегмента.

## Применённые модели:
Для решения задачи планируется использовать методы машинного обучения для классификации. Конкретные модели и их параметры будут выбраны в процессе исследования.

## Результаты:
Ожидаемые результаты включают:
- Модель с высокой точностью, способная предсказывать снижение активности клиентов.
- Сегментация клиентов на основе данных о покупательской активности и прибыльности.
- Разработка персонализированных предложений для каждого сегмента клиентов.
